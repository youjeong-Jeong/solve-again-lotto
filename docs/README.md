# 구현 기능 목록

## 입력
### 로또 구입 금액 입력 받기
---
#### 정상적인 입력
- 1,000원 단위의 입력

#### 예외 상황
- 1,000원으로 나누어 떨어지지 않는 경우
- 공백인 경우
- 문자 입력인 경우
- 소숫점 숫자인 경우

### 당첨 번호 입력 받기
---
#### 정상적인 입력
- 1~45 사이의 중복되지 않는 숫자 6개

#### 예외 상황
- 1~45 사이의 숫자가 아닌 경우
- 숫자가 중복된 경우
- 공백인 경우
- 문자 입력인 경우
- 소숫점 숫자인 경우

### 보너스 번호 입력 받기
---
#### 정상적인 입력
- 당첨 번호 6개와 중복되지 않는 수

#### 예외 상황
- 당첨번호에 포함된 수와 중복되는 경우
- 1~45 이내의 숫자가 아닐 떄

## 출력
#### 결과 출력
---
- 발행한 로또 수량 및 번호를 출력
- 당첨 내역 출력
- 수익률 출력 (소수점 둘째 자리에서 반올림);
